# TypeScript

## TypeScript是什么?

- 以JavaScript为基础构建的语言 
- 一个JavaScript的超集
- TypeScript扩展了JavaScript,并添加了类型
- 可以在任何支持JavaScript的平台中执行
- TS不能被JS解析器直接执行

## TypeScript增加了什么？

- 类型
- 支持ES的新特性
- 添加ES不具备的新特性
- 丰富的配置选项
- 强大的开发工具
