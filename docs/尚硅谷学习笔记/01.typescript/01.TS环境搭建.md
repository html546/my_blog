# TypeScript环境搭建


1. 下载Node.js
2. 安装Node.js
3. 使用npm 全局安装typescript
   1. 进入命令行
   2. 输入：npm i -g typescript
4. 创建一个ts文件
5. 使用tsc对ts文件进行编译
   1. 进入命令行
   2. 进入ts文件所在目录
   3. 执行命令：tsc xxx.ts
